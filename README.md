# TelescriptGenius

**Scripting Simplicity, Genius Complexity**

TelescriptGenius is a versatile desktop application designed to streamline script generation for telecom professionals. Developed using Python, PyQt5, and PySide (with support for PyQt), this application offers an intuitive interface to generate telecom-related scripts based on available regions. It supports multiple script templates and aims to improve efficiency in the telecom workspace.

## Features

- **Script Generation**: Easily generate telecom scripts based on different regions.
- **Template Support**: Choose from up to 3 distinct script templates.
- **Planned Features**: Future updates include a Link Check feature and additional tools to enhance the telecom workspace.

## Installation

To get started with **TelescriptGenius**, follow these steps:

### Requirements

- Python 3.x
- PyQt5 (or PySide for compatibility)
- Additional Python libraries (listed in `requirements.txt`)

### Installation Steps

1. Clone the repository:

    ```bash
    git clone https://github.com/yourusername/TelescriptGenius.git
    cd TelescriptGenius
    ```

2. Install required dependencies:

    ```bash
    pip install -r requirements.txt
    ```

3. Run the application:

    ```bash
    python main.py
    ```

## Usage

Once the application is running, follow these steps to generate your scripts:

1. **Select Region**: Choose the region for which you want to generate the script.
2. **Choose Template**: Select from the available script templates.
3. **Generate Script**: Click the button to generate the script and save it to your preferred location.

### Planned Features

- **Link Check**: A future feature to validate the connections within the telecom infrastructure.
- **Additional Tools**: Tools to further enhance productivity and simplify the telecom workspace.
